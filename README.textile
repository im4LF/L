h1. Logger

Log any messages from php into files or database

h2. Log into file

Connection code in your script:

bc. require_once('./logger/L.php');
LF('file://./logger/logs')                     // setup dsn for logger
    ->run(urldecode($_SERVER['REQUEST_URI'])); // define log key for current session

Dsn - @driver@://@path/to/logs/dir@

Write log message:

bc. L($some_var);
...
L($other_var, $again_var, 'some string');

Function @L@ have a variable-length argument lists.

To show log messages at page generated by your php script, just write this js code after @<body>@:

bc. <script type="text/javascript" src="/logger/L.js"></script>

h2. Log into database

Connection code in your script:

bc. require_once('./logger/L.php');
LF('mysql://user:pass@host/database?encoding=utf8&table=logs_table') //setup dsn for logger
    ->run(urldecode($_SERVER['REQUEST_URI']));                       // define log key for current session

Dsn is a simple dsn but with one addition - @table=logs_table@, this is a table in that log messages will be written. Table will be creating automatically if not exists.
